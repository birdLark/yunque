# 1.配置文件的配置

`
writer:
  plugin: "mysqlwriter"
  url: "hdfs://127.0.0.1:9000"
  username: "root"
  writer_path: "/my/honghu/honghu.txt"
`

# 2.参数的说明

| 参数名称 | 参数描述       |
| ----------- | -------------- |
| plugin      | 插件的名称          |
| url         | 请求访问hdfs的路径   |
| username    | 用户名             |
| writer_path | 写入hdfs文件路径    |
