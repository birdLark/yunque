# 1.配置文件的配置

`
writer:
  plugin: "mysqlwriter"
  url: "jdbc:mysql://localhost:3306/filedb?useUnicode=true&characterEncoding=utf8&serverTimezone=UTC"
  username: "root"
  password: "root"
  table: "student"
  column: "id,name"
`

# 2.参数的说明

plugin: 插件的名称
url: 请求访问的路径
username：用户名
password：密码
table：写入的表
column：写入的列
